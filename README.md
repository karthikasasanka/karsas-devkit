# Atomic Dev

A Claude Code skill for clean, incremental development.

## Install

    /plugin install https://github.com/karthikasasanka/karsas-devkit

## Usage

    /devkit:atomic <what to implement>

## Workflow

-   Write one function\
-   Write tests\
-   Run tests\
-   If green → commit\
-   If red → fix and repeat
